# Source Evaluation Matrix — NATO Admiralty Code

Two independent dimensions: **reliability of source** × **credibility of
information**. Score every source on every piece of evidence used.

## Reliability of source

| Code | Meaning           | Use                                                           |
|------|-------------------|---------------------------------------------------------------|
| A    | Completely reliable | Verified history of accuracy. Reserve for established primaries. |
| B    | Usually reliable  | Mostly accurate; occasional minor errors.                     |
| C    | Fairly reliable   | Mixed track record; sometimes accurate.                       |
| D    | Not usually reliable | Bias or sensationalism dominates; treat with caution.      |
| E    | Unreliable        | Pattern of inaccuracy. Cite only as foil.                     |
| F    | Cannot be judged  | Unknown source, no track record.                              |

## Credibility of information

| Code | Meaning                          |
|------|----------------------------------|
| 1    | Confirmed by other independent sources |
| 2    | Probably true                    |
| 3    | Possibly true                    |
| 4    | Doubtful                         |
| 5    | Improbable                       |
| 6    | Cannot be judged                 |

## Combine

Cite as `A1`, `B2`, `C3`, `F6` etc. Anything below `C3` should not anchor a
key judgment without corroboration.

## Worked example

| Item                                              | Source             | Code | Notes                              |
|---------------------------------------------------|--------------------|------|------------------------------------|
| Vessel re-routing in Strait of Hormuz             | UKMTO advisory     | A1   | Primary maritime operations source |
| IRGC drone sortie count                           | Iran International | C3   | Useful but framing-driven          |
| Brent crude move on closure rumor                 | Bloomberg wire     | A1   | Market data, traceable             |
| Houthi statement on Red Sea closure               | SABA news          | D4   | State-affiliated; treat as posture |

## Anti-patterns

- Citing a think tank that cited a primary as if it were the primary. Score
  the chain you actually have, not the chain you wish you had.
- Defaulting to F6 because you didn't bother to check the source. F6 is
  only when the source is genuinely unknown.
- Round-tripping reliability through credibility — they're independent.
