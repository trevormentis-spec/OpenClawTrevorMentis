### Key Learnings
1. **Initial cartel intelligence was stale.** My pre-pivot assumption was El Mencho alive with health problems. He was killed Feb 22, 2026. El Mayo captured July 2024. The accelerated learning session surfaced this within the first hour. This is the strongest argument for the "entity file last-source-date" framework feature.
2. **80 sources is too many for daily triage without tiering.** The Admiralty tiers solve this — A1 sources get daily scan, A2 weekly, B-sources as-needed. But the tiering is only useful if actually applied to collection decisions.
3. **The knowledge architecture schema generalizes cleanly.** actors/geography/frameworks/chronologies survived the redirect test. Structure is topic-agnostic.
4. **The postdiction mechanism is broken.** 5/0/50 split is a data artifact, not honest calibration. Fixing this is the single highest-leverage framework improvement.
5. **The huachicoleo 4-layer model works.** Tested against Salamanca seizure pattern, it correctly identified the event as simultaneously corruption, cartel-revenue, infrastructure-vulnerability, and political-risk — explaining dimensions a single-lens analysis would miss.
