API Endpoints

GET /api/briefs/latest.json

Get the most recent intelligence brief. Returns structured JSON with BLUF, theater analysis, key judgments, and prediction market data. Updated daily by 07:00 PT.

No authentication required

curl https://quiet-kangaroo-c0b94c.netlify.app/api/briefs/latest.json
GET /api/briefs/{YYYY-MM-DD}.json

Get a specific day's brief by date. Returns the same schema as latest.json.

curl https://quiet-kangaroo-c0b94c.netlify.app/api/briefs/2026-05-05.json
POST /api/subscribe

Register an agent webhook for push delivery. Requires a webhook URL and optional API key.

curl -X POST https://quiet-kangaroo-c0b94c.netlify.app/api/subscribe \
  -H "Content-Type: application/json" \
  -d '{
    "agent_name": "My Agent",
    "webhook_url": "https://my-agent.example.com/briefs",
    "api_key": "optional-agent-key",
    "tier": "agent-pro"
  }'

Agent Integration Examples

OpenClaw Agent — Pull latest brief

Any OpenClaw agent can pull the latest brief and use it as context:

curl -s https://quiet-kangaroo-c0b94c.netlify.app/api/briefs/latest.json \
  | jq '.executive_summary.bluf'

Python Agent — Parse and use intelligence

import requests, json

brief = requests.get(
    "https://quiet-kangaroo-c0b94c.netlify.app/api/briefs/latest.json"
).json()

bluf = brief["executive_summary"]["bluf"]
theaters = brief["theaters"]
judgments = brief["executive_summary"]["key_judgments"]

for theater in theaters:
    print(f"[{theater['name']}] {theater['lead'][:100]}")
    for dev in theater['developments']:
        print(f"  → {dev}")

Webhook Receiver — Auto-update on publish

from flask import Flask, request
app = Flask(__name__)

@app.route('/briefs', methods=['POST'])
def receive_brief():
    data = request.json
    brief_id = data['brief_id']
    download_url = data['download_url']
    print(f"New brief: {brief_id}")
    print(f"Download: {download_url}")
    return "OK", 200
